✅ **Funciones** – Escribe código más modular y reutilizable.  
✅ **Clases y Objetos** – Introducción a la Programación Orientada a Objetos (OOP).  
✅ **Ejercicios Desafiantes** – Retos de lógica, manipulación de datos y más.
